LDT - The plan

LDT is the leJOS Development Tool, which supports any development based on the leJOS virtual machine for LEGO Mindstorms.

Short term plan

  • Provide an integrated toolset for developing based on leJOS
  • Provide wizards for project creation
  • Compile, link and download a leJOS program
  • Support firmware installation
  • Support running in emulator
  • Provide update site at www.lejos.org, for LDT update support

Long term plan

  • Integrate leJOSUnit into LDT for JUnit based testing
  • Download other firmwares
  • Support developing leJOS for itself
  • Integrate CDT for C development
  • Support NQC, using editors
