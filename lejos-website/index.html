leJOS: Java based OS for Lego RCX

[ leJOS at SourceForge | TinyVM | API Reference ]

Like its predecessor (TinyVM), leJOS is a tiny JavaTM runtime. It functions as a replacement firmware for the Lego MindstormsTM RCX microcontroller, which is part of Lego's Robotics Invention System (RIS). The RCX contains a Hitachi H8300 processor and 32 Kb of RAM. We also have plans to port leJOS to other small devices.

TinyVM vs. leJOS

TinyVM is about 5 to 6 Kb smaller than leJOS in the RCX. The most important features missing in TinyVM which are currently supported by leJOS are:
  • Floating point operations
  • String constants
Additionally, all references in the operand stack are marked as such. This means that garbage collection can be implemented without much extra effort. We are also writing several APIs consistent with this goal (e.g. Vector and Hashtable). Other planned enhancements include: Multi-program downloading, program arguments, auto power off, a fairly complete java.lang.Math class, java.io APIs, java.net APIs, etc.

Downloading

There is no downloadable version of leJOS at this stage, but you can access the CVS repository. The module name is lejos. If you are unfamiliar with CVS, visit the SourceForge Documentation Project.

leJOS should work on any Unix system, including CygWin. Do check the README file for installation instructions. Consider the code pre-alpha and not very stable, as it's constantly getting changed.

A first release is planned for late September or October 2000.

The leJOS webpages are also on CVS. So you can not only contribute code, but also webpages and documentation. Please contact us if you would like to become part of the leJOS team!
